This folder contains app assets.

fonts/    - Add Rajdhani font files here (download from Google Fonts)
images/   - Sport thumbnails, team logos
icons/    - Sport category icons
animations/ - Lottie JSON animation files

FONTS DOWNLOAD:
Go to: https://fonts.google.com/specimen/Rajdhani
Download family and place in assets/fonts/:
- Rajdhani-Regular.ttf
- Rajdhani-Bold.ttf  
- Rajdhani-SemiBold.ttf

NOTE: If you don't add fonts, Flutter will use the default font.
Remove the fonts section from pubspec.yaml if not adding custom fonts.
